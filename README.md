# 🚔 Crime Pattern Analysis and Prediction Using Python

## 📊 Project Overview

This project focuses on analyzing real-world crime data to uncover patterns, trends, and insights using Python. The analysis includes Exploratory Data Analysis (EDA), data visualization, and basic machine learning to understand crime behavior across time, location, and categories.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze crime distribution by time (hour & month)
* Identify most frequent crime types
* Detect high-crime areas (hotspots)
* Visualize patterns using graphs and charts
* Apply Machine Learning (Linear Regression) for trend analysis

---

## 📈 Key Analysis Performed

* Crime distribution by **Month and Hour**
* Identification of **Top Crime Types**
* Victim demographic analysis
* Area-wise hotspot detection
* Correlation analysis using heatmap
* Statistical analysis (skewness, outliers)

---

## 🔍 Key Insights

* Peak crime occurs during **midday and evening hours**
* **Vehicle theft** is the most frequent crime
* Multiple **high-risk areas (hotspots)** identified
* Crime patterns are **time-dependent and non-linear**
* Weak correlations indicate **complex behavior patterns**

---

## 🤖 Machine Learning

* Applied **Linear Regression**
* Observed **no strong linear trend** in crime occurrence
* Indicates need for advanced models for better prediction
